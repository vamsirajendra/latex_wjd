## latex_wjd

This repository is a collection of LaTeX resources that I find useful.

bibtex/ MATHSCINET bibtex data for some authors in my field, and some bst style files
cls/ some documentclass files
fonts/ some documentation on selecting fonts
sty/ some style files and macros
templates/ some templates/examples for math papers, notes, and exams
tikz/ example code for generating tikz diagrams
